# SpellTech - 3D Website with CCTV Price Estimator

A modern 3D website for SpellTech featuring an enhanced auto-scrolling ad banner and CCTV price estimation tool.

## Features

- **3D Background Animation**: Interactive particle system with multiple themes
- **Enhanced Ad Banner**: Auto-scrolling ads with YAML configuration
- **CCTV Price Estimator**: Interactive calculator for CCTV installation quotes
- **Responsive Design**: Mobile-friendly layout
- **Modern UI**: Glassmorphism design with smooth animations

## Local Development

1. Clone the repository
2. Open `index.html` in a web browser
3. For full functionality, serve through a local HTTP server:
   ```bash
   python -m http.server 8000
   ```

## Configuration

### Ad Banner Configuration
Edit `ads-config.yml` to customize the ad banner content:
- Add/remove ad slides
- Change images, titles, and descriptions
- Modify auto-scroll settings

### CCTV Pricing
Pricing can be adjusted in `main.js` in the `calculateCCTVPrice()` function.
